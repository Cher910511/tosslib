# 可信开源代码库平台介绍

## 平台概述

可信开源代码库是一个面向企业级开源治理的综合性平台，提供软件库管理、组件管理、风险预警、数据大屏、AI助手等核心能力，帮助组织实现开源软件从选型、引入、追踪到安全治理的全流程闭环。

## 核心目标

平台围绕六大目标建设：合法合规、漏洞安全、软件优选、来源可信、过程透明、全生命周期管理。

## 业务总流程

平台业务从引入选型开始，全流程进行风险感知预警，实时获取风险信息及补丁修复方案。基于软件信息、SBOM、风险感知信息等内容形成可信版本火车信息树，技术评估确认后形成初版可信软件并入库，最终产出可信开源版本火车计划，持续进行治理。

## 平台服务能力

平台面向重点行业运营团队、研发团队、行业主管部门、软件研发企业及个人开发者，提供六大类服务：

- 可信咨询服务：开源使用摸底、治理成熟度评估、开源组织建设、风险应对规划
- 可信资产服务：可信资产供应、资产更新、定制化软件治理、软件优选
- 可信治理工具服务：可信软件中心仓、研发流程对接、代码扫描、制品扫描
- 研发工具链部署：代码托管、智能协作开发、持续集成与交付
- 风险预警服务：软件成分分析结果回传、在用开源软件风险预警
- 能力平台：可信软件选型（行业库、检索、推荐）、可信治理报告、智能问答、组织管理
