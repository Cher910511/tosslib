# 个人设置

个人设置模块帮助您统一管理个人资料、订阅资产及预警通知配置。

## 1. 进入个人设置

在系统任意页面中，点击或悬停右上角的用户头像或昵称区域，在下拉菜单中选择 **个人设置**，即可进入个人设置页面。

![personal-settings-1](../images/personal-settings-1.png)

## 2. 个人信息管理

在个人信息标签页中，您可以查看并维护个人账户基础信息。支持修改头像、昵称、邮箱、手机号、密码。

![personal-settings-2](../images/personal-settings-2.png)

## 3. 订阅列表

订阅列表用于集中管理您订阅的软件与组件资产。

* 支持查看已订阅的软件名称、软件版本、发布时间、行业分类、开发商及编程语言等信息；
* 支持通过顶部搜索框快速检索指定软件；如无需继续关注，可点击操作列中的 **「取消订阅」** 进行移除。

> 订阅后，当软件产生漏洞或风险时，系统将自动生成对应预警信息。

![personal-settings-3](../images/personal-settings-3.png)

## 4. 预警信息

预警信息模块用于统一展示您所订阅或选型的软件风险告警。

当软件检测到漏洞、恶意代码或其他安全风险时，系统将自动生成对应预警记录。

支持展示以下信息：
* **预警时间** — 记录风险发现时间；
* **预警标题** — 展示受影响的软件及关联漏洞编号；
* **预警概要** — 展示漏洞命中规则与风险描述；
* **处理状态** — 标识当前是否已处理；
* **置信度** — 展示检测结果可信程度。

点击标题可查看漏洞预警详情，修改该漏洞预警的处理状态，对于已确认或处理完成的预警，可通过右侧操作按钮进行删除。

![personal-settings-4](../images/personal-settings-5.png)
![personal-settings-4](../images/personal-settings-6.png)

## 5. 预警信息设置

预警信息设置用于配置风险通知的接收方式，确保在出现漏洞或安全事件时能够及时收到提醒。支持以下通知渠道：

**邮箱通知**：勾选「邮箱通知」后启用邮箱告警，支持填写一个或多个接收邮箱，可通过 **「+ 添加新邮箱」** 增加多个通知地址。

**短信通知**：勾选「短信通知」后启用短信提醒，支持填写一个或多个手机号，可通过 **「+ 添加新短信」** 添加多个接收号码。

![personal-settings-7](../images/personal-settings-7.png)

> 提示：建议定期检查通知配置与联系方式，确保能够及时接收到最新风险预警信息。
