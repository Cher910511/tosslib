# AI助手使用指南

## 基本用法

### 1. 发起提问

在输入框中输入您的问题后发送，AI 助手将自动结合平台数据进行检索与分析，并返回对应结果。在正式生成回答前，系统会展示 AI 的思考与分析过程，帮助用户了解当前问题的检索逻辑与推理路径。

![assistant-1](../images/assistant-1.png)

### 2. 查看推荐结果

当 AI 助手推荐相关软件或组件时，回答内容中会展示对应的软件 / 组件卡片。点击卡片后，可直接跳转至平台详情页，进一步查看软件/组件的详细信息。

![assistant-2](../images/assistant-2.png)

## 提问示例

### 软件发现与选型

- 推荐几个支持国产化麒麟系统的Java加密库
- 我想找一个类似Fastjson但更安全的JSON处理工具
- 目前平台中最火的前端框架有哪些？

### 安全合规评估

- Log4j 2.15.0版本现在还有高危漏洞吗？
- 这个组件采用的是什么协议？可以商用吗？
- 帮我对比一下A版本和B版本的安全性

### 集成与技术细节

- 如何在Maven中引入ECharts组件？
- 这个组件依赖了哪些上游包？有冲突风险吗？
- 源码下载地址在哪里？我想看下实现

### 溯源与维护可信度

- 这个软件是谁开发的？是个人还是大公司？
- 这个项目还在维护吗？最后一次更新是什么时候？

## 建议提问方式

为了获得更准确的结果，建议在提问时尽量描述：

- 软件名称
- 版本号
- 使用场景
- 技术栈
- 操作系统
- 目标需求

例如：推荐一个支持 Spring Boot 3、国产化环境可用、License 风险较低的工作流引擎。

# 温馨提示

- AI回答基于平台当前收录的数据生成；
- 部分社区动态与漏洞信息可能存在同步延迟；
- 对于关键生产环境决策，建议结合人工进一步验证；
- AI生成内容仅供参考，请结合实际业务场景使用；
