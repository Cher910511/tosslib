# 组织详情

## 进入组织详情

在组织列表中点击任意组织卡片后，即可进入对应的组织详情页。页面顶部展示组织名称与组织描述，下方包含多个功能标签页，包括：组织信息、组织成员、成员权限、选型列表、订阅列表、预警信息、预警信息设置及需求反馈等。

组织详情页是组织级资产与权限管理的核心页面，组织创建者及管理员可在此统一管理组织成员、软件订阅、风险预警及需求反馈等内容。


## 组织信息

组织信息模块用于维护组织基础资料，并支持企业认证申请。基础信息包括：组织名称、组织描述，支持对组织基础信息进行编辑。

![org-detail-1](../images/org-detail-1.png)

### 企业认证申请

组织可通过企业认证提升组织可信度与标识能力。申请流程如下：

1. 点击「申请企业认证」按钮；
2. 填写企业名称、联系电话、统一社会信用代码、法人姓名等信息，并上传营业执照；
3. 提交后等待后台审核，预计 1～3 个工作日完成。

![org-detail-2](../images/org-detail-2.png)

> 说明：同一企业支持多个账号发起认证申请，但仅允许认证成功一次，企业认证结果不可转移；认证成功后，组织名称旁将展示企业认证标识。

## 组织成员

组织成员模块用于管理组织内的成员与邀请链接。成员列表展示组织内所有成员，并标识组织创建者。

支持：
* 按成员名称搜索；
* 退出组织或移除普通成员（需具备对应权限）。

### 邀请成员

邀请流程如下：

1. 点击「创建邀请链接」；
2. 设置链接失效时间（默认 7 天）；
3. 填写备注信息（可选，最多 256 字）；
4. 创建后复制邀请链接并发送给待邀请成员。

![org-detail-3](../images/org-detail-3.png)

## 成员权限

组织支持配置多个管理员，管理员拥有组织最高权限，可管理成员权限、修改组织信息、管理版本火车等。管理员不可取消自身管理员身份，不可直接退出组织，但可将管理员身份转移给其他成员，转移后自动降级为普通成员。

成员权限模块用于配置组织内不同角色的功能访问权限，修改权限后，点击保存即可生效：

支持配置的权限项包括：

| 权限项 | 说明 |
|---|---|
| 订阅软件 | 是否允许将软件订阅到组织中 |
| 编辑组织信息 | 是否允许编辑组织信息及申请企业认证 |
| 管理组织成员 | 是否允许管理邀请链接及移除成员 |
| 管理成员权限 | 是否允许查看并管理成员权限模块 |
| 需求反馈列表 | 是否允许查看需求反馈标签页 |
| 需求反馈导出 | 是否允许导出需求反馈数据 |
| 查看版本火车 | 是否允许查看组织版本火车列表 |
| 查看订阅列表 | 是否允许查看组织订阅列表 |
| 预警信息设置 | 是否允许配置组织预警通知 |
| 查看预警信息 | 是否允许查看组织漏洞预警信息 |

![org-detail-4](../images/org-detail-4.png)

## 订阅列表

订阅列表用于集中管理组织订阅的软件与组件资产。

* 支持查看已订阅的软件名称、软件版本、发布时间、行业分类、开发商及编程语言等信息；
* 支持通过顶部搜索框快速检索指定软件；如无需继续关注，可点击操作列中的 **「取消订阅」** 进行移除。

![org-detail-6](../images/org-detail-6.png)

> 订阅后，当软件产生漏洞或风险时，系统将自动生成对应预警信息。

### 预警信息

预警信息模块用于统一展示组织所订阅或选型的软件风险告警。

当软件检测到漏洞、恶意代码或其他安全风险时，系统将自动生成对应预警记录。

支持展示以下信息：
* **预警时间** — 记录风险发现时间；
* **预警标题** — 展示受影响的软件及关联漏洞编号；
* **预警概要** — 展示漏洞命中规则与风险描述；
* **处理状态** — 标识当前是否已处理；
* **置信度** — 展示检测结果可信程度。

点击标题可查看漏洞预警详情，修改该漏洞预警的处理状态，对于已确认或处理完成的预警，可通过右侧操作按钮进行删除。

## 预警信息设置

预警信息设置用于配置风险通知的接收方式，确保在出现漏洞或安全事件时能够及时收到提醒。支持以下通知渠道：

**邮箱通知**：勾选「邮箱通知」后启用邮箱告警，支持填写一个或多个接收邮箱，可通过 **「+ 添加新邮箱」** 增加多个通知地址。

**短信通知**：勾选「短信通知」后启用短信提醒，支持填写一个或多个手机号，可通过 **「+ 添加新短信」** 添加多个接收号码。

![org-detail-9](../images/org-detail-9.png)

## 需求反馈

需求反馈模块用于统一查看组织成员提交的问题反馈与需求建议。

列表展示内容包括：联系方式、反馈类型、反馈内容、提交时间；

支持按手机号、邮箱及反馈类型进行筛选。点击「查看详情」后，可查看完整反馈内容。支持导出反馈列表
