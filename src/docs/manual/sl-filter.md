# 软件筛选

## 页面概述

进入 **软件库** 页面后，可集中查看平台收录的开源软件资产及其可信、安全与治理信息。页面集成了多维筛选、漏洞风险识别、开发商溯源及分页浏览等能力，帮助研发、安全与合规人员高效完成开源软件的检索、评估与管理。

注：
- 普通成员需要被授权后，才可在软件库中检索和查看所在组织中已通过版本火车发布的软件。
- 组织管理员可查看平台的全部软件及组件，不受版本火车范围限制。

## 搜索与筛选

页面顶部提供统一搜索与多条件筛选入口，支持通过组合条件对开源软件进行精准定位与批量过滤。

支持的筛选条件包括：

- **软件名称**：支持按软件名称进行检索，例如 `requests`、`Spring Boot` 等。

- **软件版本**：支持根据指定版本号进行筛选，便于定位特定版本的软件资产与风险情况。

- **编程语言**：支持按主流开发语言筛选，如 `Python`、`Java`、`JavaScript` 等，快速聚焦对应技术栈。

- **漏洞状态**：支持按“有漏洞”或“无漏洞”进行过滤，快速识别存在安全风险的软件。

- **行业分类**：支持按工业、通信等行业场景进行分类筛选，便于结合业务领域进行检索。

- **版本火车**：支持筛选所在组织的所有已发布版本火车。

## 软件列表

页面中心区域以结构化列表形式展示软件资产信息，帮助用户快速了解软件的可信度、安全性与基础元数据。

列表字段说明如下：

| 字段 | 说明 |
|---|---|
| 软件名称 | 点击后可进入软件详情页，查看软件详细信息与版本数据。 |
| 最新版本评分 | 展示软件最新版本的 Scorecard 安全评分，用于衡量软件安全治理成熟度。 |
| 版本发布时间 | 软件最新版本的发布日期。 |
| 行业分类 | 标识软件所属行业或应用领域。 |
| 最新版本漏洞数 | 展示当前最新版本已收录的 CVE 漏洞数量。 |
| 开发商 | 展示软件所属组织或主要维护方。 |
| 编程语言 | 展示软件主要使用的开发语言。 |
