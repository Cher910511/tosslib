## 概述

进入 **可信开源代码库** 首页后，可快速完成开源软件的搜索、发现、风险感知与资产管理。首页聚合了软件检索、推荐内容、漏洞告警及分类筛选等核心能力，帮助团队更高效地管理开源资产。


## 全局搜索

**页面中心区**提供统一搜索入口，支持按不同对象类型进行筛选：

- **软件** — 支持根据软件名称、版本、漏洞状态、编程语言、行业分类等条件进行搜索，并进入对应的软件库列表页。
- **组件** — 支持根据组件名称、版本及编程语言等信息进行检索，并进入组件库列表页。

> 平台将自动保存搜索历史，便于后续快速回溯与再次检索。

## 推荐软件

**首页左侧区域**展示「推荐软件」卡片列表，用于快速浏览平台推荐的软件项目。每张卡片包含：

| 字段 | 说明 |
|------|------|
| 软件名称与版本 | 软件唯一标识及当前推荐版本 |
| 编程语言 | 软件主要编程语言 |
| 发布日期 | 版本的发布时间 |
| 开发商 | 软件所属组织或主要维护者 |

## 漏洞信息告警

通过订阅软件，首页可实时聚合展示**最新漏洞动态**，支持查看漏洞标题、风险摘要及发布时间等关键信息。

- 支持区分“个人订阅”与“组织订阅”的漏洞告警；

- 点击告警信息，可快速进入对应的漏洞预警列表，查看详细风险内容与影响范围。

> 建议订阅所有使用中的软件，以便在风险发生时第一时间获取漏洞预警与变更通知。
