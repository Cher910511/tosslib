# 组织列表

组织列表展示当前用户所属的所有组织。

列表信息：
每个组织卡片包含组织名称、组织简介，点击卡片可进入该组织详情页。

搜索组织：
在搜索框中输入组织名称关键词，可快速筛选目标组织。

## 新建组织

所有注册用户均可自由创建组织。操作步骤：
1. 点击组织列表右上角 **新增组织** 按钮；
2. 在弹出的表单中填写组织名称、组织介绍，点击 **创建** 按钮完成操作；

![organization-1](../images/organization-1.png)

## 删除组织

权限说明：删除组织需要特定的权限，普通成员无法执行删除操作。如果您拥有删除权限，组织卡片上会显示 **删除组织** 按钮。

![organization-2](../images/organization-2.png)

注意事项：
- 删除操作不可恢复，组织内的所有软件、成员数据将一并清除，建议在删除前备份重要数据；
- 如无删除权限，请联系组织管理员处理；
